# اختبار تفاعلي ذكي لمقرر إنتاج برامج الفيديو

هذه نسخة منشورة عبر GitHub Pages لاختبار مراجعة مقرر **إنتاج برامج الفيديو**. تتضمن:

- بنك أسئلة من 300 سؤال.
- إدخال اسم الطالب قبل بدء الاختبار.
- ترتيب عشوائي للأسئلة.
- خلط بدائل الاختيار من متعدد.
- نمط تكيفي يرفع أو يخفض الصعوبة وفق أداء الطالب.
- تغذية راجعة فورية.
- علامة مائية باسم الطالب ووقت الدخول.
- تعطيل النسخ والطباعة والنقر الأيمن من واجهة الاختبار.
- تقرير نهائي قابل للطباعة والتصدير CSV.

## الروابط بعد النشر

رابط الطلاب:

`https://drwaheed20-bot.github.io/video-production-smart-quiz/`

لوحة التحكم:

`https://drwaheed20-bot.github.io/video-production-smart-quiz/admin.html`

## التحكم في التفعيل والإيقاف

لأن GitHub Pages استضافة ثابتة، يتم التحكم العام من ملف:

`settings.json`

لتفعيل الاختبار:

```json
"enabled": true
```

لإيقاف الاختبار:

```json
"enabled": false
```

بعد تعديل الملف وعمل Commit، يتغير وضع الاختبار خلال فترة قصيرة.

## حدود الحماية

لا توجد صفحة ويب تمنع تصوير الشاشة أو تحميل الملفات بنسبة 100%، خصوصًا إذا كان المستودع عامًا. هذه النسخة تقلل التسريب من خلال العلامة المائية وتعطيل النسخ والطباعة وخلط الأسئلة والبدائل. للحماية الأقوى والنتائج المركزية، يلزم تحويل المنصة لاحقًا إلى خادم فعلي أو Cloudflare Worker/Pages Functions أو Firebase.
